# Portfólio Técnico - Infraestrutura, Automação e Desenvolvimento Aplicado

## Apresentação

Este portfólio reúne projetos, estudos aplicados e arquiteturas técnicas construídas para demonstrar domínio prático em infraestrutura de TI, automação operacional, suporte técnico, integração de sistemas e desenvolvimento aplicado. A proposta é apresentar capacidade real de transformar demandas técnicas e rotinas repetitivas em soluções organizadas, documentadas, reutilizáveis e seguras.

## Síntese Profissional

Este repositório apresenta um conjunto curado de projetos técnicos que evidenciam capacidade de atuar em diferentes camadas da operação de TI: infraestrutura, suporte, automação, observabilidade, integração de sistemas e desenvolvimento aplicado. Os cases foram estruturados para mostrar raciocínio técnico, organização de solução, domínio de ferramentas e capacidade de transformar necessidades operacionais em entregas reutilizáveis.

## Áreas de Atuação

- Infraestrutura Windows Server
- Active Directory, DNS, DHCP e GPO
- Monitoramento com Zabbix e Grafana
- Automação de suporte com Python e PowerShell
- Docker, WSL2 e ambientes Linux
- Integração de sistemas e atendimento omnichannel
- Desenvolvimento de extensões de navegador
- Automação documental e relatórios
- Publicação web e presença digital

## Projetos em Destaque

- [Extensões de Navegador](cases/01-extensoes-navegador.md): desenvolvimento de extensões Chrome com foco em privacidade visual, produtividade local, controle de interface e organização de navegação.
- [Laboratório WSL2, Ubuntu e Docker](cases/02-wsl-ubuntu-docker-lab.md): estruturação de ambiente híbrido para serviços Linux em Windows, containers, banco de dados e troubleshooting.
- [Programa de Suporte Técnico](cases/03-programa-suporte-tecnico.md): aplicação desktop para centralizar rotinas de suporte Windows com interface Python e automação local.
- [Sistema de Controle de Processos e Relatórios](cases/04-sistema-controle-processos-relatorios.md): automação administrativa com modelagem relacional, scripts Python e apoio adicional em VBA e Excel.
- [Ambiente Windows Server Corporativo](cases/05-windows-server-corporativo.md): arquitetura narrada de infraestrutura Microsoft com domínio, políticas, serviços de rede e governança técnica.
- [Chatwoot com IA e Atendimento Omnichannel](cases/06-chatwoot-ia-omnichannel.md): arquitetura documentada de atendimento omnichannel com automação de resposta e integração de canais.
- [Observabilidade com Zabbix e Grafana](cases/07-observabilidade-zabbix-grafana.md): trilha técnica de monitoramento, dashboards e alertas com integração entre observabilidade e operação.
