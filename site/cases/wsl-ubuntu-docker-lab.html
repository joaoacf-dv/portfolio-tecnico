Laboratório WSL2, Ubuntu e Docker

Ambiente híbrido para serviços Linux em Windows com containers, banco de dados e troubleshooting de instalação.

Laboratório Técnico
WSL2 Ubuntu Docker Docker Compose MySQL

Visão Geral

O laboratório documenta a preparação de um ambiente Linux sobre host Windows para subir serviços locais com containers, testar integrações e registrar procedimentos de troubleshooting.

Desafio Técnico

Era necessário estruturar um ambiente reproduzível para serviços Linux e banco de dados em Windows, com isolamento suficiente para testes, ajustes de dependência e validação técnica sem depender de infraestrutura externa.

Solução Desenvolvida

O ambiente foi organizado com WSL2 e Ubuntu como camada base, Docker e Compose para orquestração local e banco MySQL para apoiar serviços dependentes. A apresentação pública foi mantida em nível de laboratório técnico.
