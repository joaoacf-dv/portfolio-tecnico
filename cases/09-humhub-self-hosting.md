# HumHub Self-hosting

Tipo de apresentação: Microcase  

## Visão Geral

Microcase de implantação de aplicação web self-hosted com HumHub, Apache, PHP e MariaDB, apresentado como laboratório técnico de stack LAMP.

## Desafio Técnico

O cenário técnico envolve disponibilizar uma aplicação web self-hosted com os componentes essenciais de servidor web, PHP, banco relacional e rotinas básicas de operação e manutenção.

## Solução Desenvolvida

O roteiro se organiza em atualização do sistema, instalação de Apache e PHP, configuração de extensões, preparação do MariaDB, criação de banco e publicação da aplicação com diretório web e parâmetros do servidor. A narrativa foi condensada para destacar a arquitetura do ambiente e a lógica de implantação.

## Stack Principal

- HumHub
- Apache
- PHP
- MariaDB
- Cron
- Linux

## Entregas Técnicas

- Preparação de stack LAMP.
- Configuração de aplicação social self-hosted.
- Organização de diretório web e permissões.
- Base para manutenção e operação local.

## Competências Demonstradas

- Self-hosting de aplicação web.
- Configuração de Apache e PHP.
- Preparação de banco de dados.
- Operação básica de serviço Linux.

## Curadoria Pública

O microcase não publica usuários, senhas, domínios, caminhos reais ou scripts de acesso do ambiente original. O material foi reduzido a uma narrativa técnica curada.

## Resultado Técnico

O conteúdo demonstra capacidade de instalar e organizar uma aplicação web self-hosted em infraestrutura básica, com valor de portfólio como microcase de implantação.
